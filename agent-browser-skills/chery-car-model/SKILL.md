# 奇瑞汽车车型查询

## 安装步骤

在使用此技能之前，请确保已安装必要的依赖包：

```bash
python -m pip install wuying-agentbay-sdk
```

## 使用场景

- 用户想查询奇瑞汽车的车型信息
- 用户需要了解某款车型的详细配置参数
- 用户想查看车型的价格和技术规格
- 用户需要对比不同车型的配置差异

## 使用方法

```bash
python browser-use.py "<任务执行步骤>"
```

## 快速示例

```bash
python browser-use.py " \
1. 前往奇瑞汽车官网 https://www.chery.cn/ \
2. 等待页面完全加载 \
3. 点击'进入奇瑞' \
4. 等待页面跳转完成 \
5. 鼠标悬停在'全部车型'，开启下拉选项 \
6. 等待下拉菜单显示 \
7. 点击'瑞虎8系' \
8. 等待车型页面加载 \
9. 点击'了解详情' \
10. 点击'查看配置' \
11. 点击'详细配置' \
12. 等待配置信息加载完成 \
13. 查看页面中所有配置信息 \
14. 以markdown格式返回车型配置详情，包括基本参数、动力系统、底盘悬挂、安全配置、智能科技等信息 \
"
```

## 注意事项

- 始终注明信息来源为奇瑞汽车官网
- 不需要创建新的脚本，用skill目录下的browser-use.py
- 鼠标悬停操作需要等待下拉菜单完全显示
- 页面跳转和配置信息加载可能需要时间，请耐心等待
- 不同车型的配置页面结构可能略有差异
- 车型配置和价格信息可能实时变化，以查询时刻为准
- 任务需要执行1~2分钟，不要杀进程，请耐心等待和观察任务
- 可以根据实际需要查询不同的车型系列（如瑞虎8系、瑞虎9、艾瑞泽8等）
