# 安居客租房信息检索

## 安装步骤

在使用此技能之前,请确保已安装必要的依赖包:

```bash
python -m pip install wuying-agentbay-sdk
```

## 使用场景

- 用户想在指定城市查找租房信息
- 用户需要按区域、价格、房型等条件筛选房源
- 用户想查找临近地铁、电梯房等特定条件的房源
- 用户需要获取符合条件的房源列表

## 使用方法

```bash
python browser-use.py "<任务执行步骤>"
```

## 快速示例

```bash
python browser-use.py " \
1. 前往安居客网站 https://www.anjuke.com/ \
2. 等待页面完全加载 \
3. 鼠标悬停在地图标志上,开启下拉选项 \
4. 点击哈尔滨 \
5. 等待页面跳转完成 \
6. 点击'租房'选项 \
7. 等待租房页面加载 \
8. 在筛选框中位置点击'松北',租金点击'800-1000' \
9. 房型点击'一室' \
10. 勾选'近地铁'和'电梯房'选项 \
11. 等待筛选结果加载完成 \
12. 查看页面中所有符合条件的房源信息 \
13. 以markdown格式返回房源列表,包括房源标题、价格、位置、面积、标签等信息 \
"
```

## 注意事项

- 始终注明信息来源为安居客
- 不需要创建新的脚本,用skill目录下的browser-use.py
- 鼠标悬停操作可能需要等待下拉菜单显示
- 筛选条件设置后需要等待页面刷新加载结果
- 如果页面加载较慢,请耐心等待
- 房源信息可能实时变化,以查询时刻为准
- 任务需要执行1~2分钟,不要杀进程,请耐心等待和观察任务
- 可以根据实际需要调整城市、区域、价格范围、房型等筛选条件
- skill调用后，控制台会打印出asp流化链接（可视化的url），可告知用户查看
