# 华帝世界期刊查询

## 安装步骤

在使用此技能之前,请确保已安装必要的依赖包:

```bash
python -m pip install wuying-agentbay-sdk
```

## 使用场景

- 用户想查看华帝世界期刊的最新一期内容
- 用户想了解华帝品牌的期刊杂志
- 用户需要浏览华帝世界杂志的详细页面

## 使用方法

```bash
python browser-use.py "<任务执行步骤>"
```

## 快速示例

```bash
python browser-use.py " \
1. 前往华帝官网 https://www.vatti.com.cn/ \
2. 等待页面完全加载 \
3. 鼠标悬停在'华帝品牌'菜单,开启下拉栏选项 \
4. 点击下拉栏选项中的'华帝世界' \
5. 等待华帝世界页面加载完成 \
6. 点击时间最新一期的刊物,进入详情页面 \
7. 查看每一页的页面信息,逐页浏览所有内容 \
8. 以markdown格式返回期刊的完整内容信息,包括期号、标题、各页内容摘要 \
"
```

## 输出格式

```markdown
## 华帝世界期刊 - 最新一期

### 期刊信息
- 期号: 第XX期
- 发布时间: XXXX年XX月
- 标题: XXXXXXX

### 内容概览

#### 第1页
- 内容摘要...

#### 第2页
- 内容摘要...

### 统计信息
- 总页数: XX页
- 主要栏目: XXX
```

## 注意事项

- 始终注明信息来源为华帝官网
- 不需要创建新的脚本,用skill目录下的browser-use.py
- 鼠标悬停操作可能需要等待下拉菜单显示
- 期刊页面可能需要翻页查看完整内容
- 如果页面加载较慢,请耐心等待
- 任务需要执行1~2分钟,不要杀进程,请耐心等待和观察任务，也不要重试
- skill调用后，控制台会打印出asp流化链接（可视化的url），可告知用户查看
