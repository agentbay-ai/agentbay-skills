# 飞利浦剃须刀售后服务网点查询

## 安装步骤

在使用此技能之前，请确保已安装必要的依赖包：

```bash
python -m pip install wuying-agentbay-sdk
```

## 使用场景

- 用户想查询飞利浦剃须刀售后服务网点
- 用户需要在指定省市区查找附近的售后网点
- 用户想了解飞利浦在特定地区的维修服务点

## 使用方法

```bash
python browser-use.py "<任务执行步骤>"
```

## 快速示例

```bash
python browser-use.py " \
1. 前往飞利浦中国官网 https://www.philips.com.cn/ \
2. 等待页面完全加载 \
3. 滚动到页面底部 \
4. 点击'售后服务'链接 \
5. 等待页面跳转完成 \
6. 点击'售后网点'选项 \
7. 等待网点查询页面加载 \
8. 在产品类型中选择'剃须刀' \
9. 在地区筛选中依次选择'浙江省'、'温州市'、'鹿城区' \
10. 等待查询结果加载完成 \
11. 查看页面中所有符合条件的售后网点信息 \
12. 以markdown格式返回网点列表，包括网点名称、地址、联系电话、服务类型等信息 \
"
```

## 注意事项

- 始终注明信息来源为飞利浦中国官网
- 不需要创建新的脚本，用skill目录下的browser-use.py
- 页面加载可能需要时间，请耐心等待
- 滚动到底部时需要确保底部元素已加载
- 地区选择需要按照省-市-区的顺序依次选择
- 网点信息可能实时变化，以查询时刻为准
- 任务需要执行1~2分钟，不要杀进程，请耐心等待和观察任务
- 可以根据实际需要调整产品类型和查询地区
